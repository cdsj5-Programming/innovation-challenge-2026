# 示例作品摘要

以下示例根據參考範本整理，方便學生理解寫法與完整度。

## 作品名稱

單車安全監察系統

## 問題說明示例

澳門街道狹窄且人車密集，居民長期依賴電單車出行。若希望推動更環保的單車出行，首先要解決騎行安全與轉向提示不足的問題，否則市民即使有環保意識，也未必願意改用單車。

## 設計理念示例

單車是零排放的綠色交通方式，能減少碳排放與空氣污染，也有助提升市民健康。不過在車流密集的城市環境中，傳統手動打燈或以手勢示意轉向，容易令騎士分心。我們希望設計一個更安全、更直觀的方式，讓騎士在專注路面的同時，仍能清楚向周圍車輛和行人表達轉向意圖。

## 解決方案細節示例

- 以智能頭盔作為核心產品
- 頭盔配備感應器、陀螺儀與 LED 顯示模組
- 騎士只需輕微向左或向右傾頭，系統便會自動偵測動作
- 後方 LED 矩陣會顯示方向燈約 10 秒
- 同步發出提示音，提升辨識度
- 目標是減少分心操作，增加城市騎行安全
