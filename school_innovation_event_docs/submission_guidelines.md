# 第一屆聖五中學生創意設計挑戰賽提交規範

## 一、提交方式

- 每組 2 至 3 人
- 由 1 位組員代表在線上提交
- 文字內容直接填入網站
- 科學海報另外上傳 1 份 JPG

## 二、必填欄位

| 欄位 | 要求 |
|---|---|
| 班級 | 必填 |
| 組員 | 必填，需與班級一致 |
| 作品名稱 | 必填 |
| 問題說明 | 必填，200 字內 |
| 設計理念 | 必填，500 字內 |
| 解決方案細節 | 必填，1000 字內 |
| 人工智能使用情況 | 必填，建議 150 至 300 字內 |

## 三、科學海報要求

| 項目 | 規格 |
|---|---|
| 數量 | 1 份 |
| 格式 | JPG |
| 內容 | 題目、問題、設計理念、解決方案重點、預期價值 |
| 作用 | 用圖像與重點文字總結作品 |

## 四、建議撰寫方式

### 問題說明

- 直接說明你們觀察到的真實問題
- 要清楚指出問題出現在哪裡、影響誰
- 不要只寫大口號

### 設計理念

- 說明為何想到這個方向
- 可交代設計原則、核心想法、希望達成的效果
- 可以提及為何比現有做法更合適

### 解決方案細節

- 說清楚作品如何運作
- 建議寫出流程、功能、使用情境
- 可說明材料、技術、成本、風險或實施步驟

### 人工智能使用情況

- 如有使用 AI，請說明用途與平台
- 例如：資料整理、構思靈感、圖片生成、文句修飾
- 如無使用，請直接填寫「本作品未有使用 AI 工具」

## 五、提交前檢查

- [ ] 題目名稱已填
- [ ] 問題說明未超出 200 字
- [ ] 設計理念未超出 500 字
- [ ] 解決方案細節未超出 1000 字
- [ ] AI 使用情況已如實填寫
- [ ] 科學海報已匯出為 JPG
- [ ] 海報內容與文字欄位一致

## 六、常見問題

| 情況 | 做法 |
|---|---|
| 還未完成完整作品 | 先清楚寫出概念與做法 |
| 沒有使用 AI | 如實填寫即可，不必硬加 |
| 海報不是 JPG | 先轉成 JPG 再提交 |
| 提交時出現技術問題 | 立即通知老師 |
