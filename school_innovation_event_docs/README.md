# school_innovation_event_docs

## 一、用途

這個資料夾整理「第一屆聖五中學生創意設計挑戰賽」的活動文件，讓老師、導師、學生與 AI 評改流程使用同一套規格。

## 二、文件總覽

| 文件 | 用途 |
|---|---|
| `student_brief.md` | 學生版簡章 |
| `submission_guidelines.md` | 學生提交規範 |
| `example_submission.md` | 作品介紹與科學海報示例摘要 |
| `mentor_checklist_before_after.md` | 導師課前課後檢查清單 |
| `grading_system_plan.md` | AI 評改規劃 |
| `complete_workflow.md` | 三部分整體工作流程 |
| `README.md` | 本資料夾導覽 |

## 三、核心統一規格

- 活動名稱：第一屆聖五中學生創意設計挑戰賽
- 比賽時間：2026 年 4 月 18 日 10:00 至 11:30
- 參賽形式：2 至 3 人小組
- 提交內容：
  - 線上表單文字內容
  - 科學海報 1 份，格式為 JPG
- 文字欄位：
  - 問題說明
  - 設計理念
  - 解決方案細節
  - 人工智能使用情況
- AI 評改用途：
  - 初審評分
  - 資訊科技科其中一項分數參考

## 四、建議使用順序

1. 先看 `student_brief.md`
2. 再看 `submission_guidelines.md`
3. 需要示例時看 `example_submission.md`
4. 導師活動前後使用 `mentor_checklist_before_after.md`
5. AI 與老師評分流程看 `grading_system_plan.md`
6. 系統整體安排看 `complete_workflow.md`
