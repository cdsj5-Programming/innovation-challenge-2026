# 第一屆聖五中學生創意設計挑戰賽導師清單

## 一、課前準備

- [ ] 確認班級、課室與比賽時間
- [ ] 確認學生分組名單
- [ ] 測試學生說明網站、提交網站與 QR Code
- [ ] 確認 Google Drive / Google Sheet 正常
- [ ] 確認科學海報 JPG 可正常上傳
- [ ] 準備備援收件方式

## 二、入班前最後檢查

- [ ] 投影正常
- [ ] Wi-Fi / 網絡正常
- [ ] 學生能打開說明網站
- [ ] Web App 可成功提交
- [ ] 計時工具已準備

## 三、活動進行中

- [ ] 說明比賽宗旨、時間與提交要求
- [ ] 提醒必交為文字欄位加科學海報 JPG
- [ ] 提醒 AI 使用情況必須如實填寫
- [ ] 於第 30、60、80 分鐘提醒時間
- [ ] 處理登入、網絡與上傳問題

## 四、活動結束後

- [ ] 確認各組已提交
- [ ] 跟進未交或上傳失敗個案
- [ ] 核對班別、組員、題目與海報

## 五、課後整理

- [ ] 檢查 Drive 是否齊件
- [ ] 檢查 Sheet 收件紀錄
- [ ] 標記缺件、重複提交、檔案異常
- [ ] 啟動 AI 初審流程
- [ ] 備份收件資料

## 六、備援方案

| 情況 | 處理方式 |
|---|---|
| 網絡不穩 | 先完成內容，再用老師指定方式補交 |
| 表單失效 | 改用共享資料夾或電郵 |
| 海報無法匯出 JPG | 先交原始檔，由老師協助轉檔 |
| 圖片打不開 | 要求學生重新匯出並補交 |
